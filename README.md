# Word-Lernbegleiter – Wissenschaftliche Projektarbeit (Mechatronik)

Interaktiver Lernbegleiter (Single-File-Web-App) für die Unterrichtseinheit
*„Wissenschaftliche Projektarbeit in Microsoft Word"* an der Fachschule für Mechatronik.

Die Schüler verwandeln in 11 aufeinander aufbauenden Stufen einen unformatierten
Fließtext in eine fertig formatierte wissenschaftliche Projektarbeit
(Thema: *Potenziale der KI für die vorausschauende Instandhaltung mechatronischer Anlagen*).

## Veröffentlichung

Die App ist eine einzelne `index.html` ohne externe Abhängigkeiten und wird über
**GitHub Pages** bereitgestellt:

➡️ https://schwibach.github.io/word-lernbegleiter/

Der Fortschritt wird im Browser (`localStorage`) gespeichert und kann über eine
**Chiffre** bzw. einen Deeplink (`?code=…`) auf anderen Geräten fortgesetzt oder der
Lehrkraft übermittelt werden.

## Begleitmaterial

Rohfassung, Musterlösung, Abbildungen und Lehrkraft-Handreichung liegen im
SharePoint-Team *Unterricht* unter `General/Word/`.
